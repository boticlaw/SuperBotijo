## Key Effects

- Minimal glow (`text-shadow: 0 0 10px`)
- Dark-to-light transitions
- Low white emission
- High readability
- Visible focus states

---

## Anti-patterns to Avoid

- ❌ Slow updates + No automation
- ❌ Emojis as icons (use SVG: Heroicons/Lucide)
- ❌ Missing `cursor-pointer` on clickable elements
- ❌ Hover states without smooth transitions (150-300ms)
- ❌ Light mode with text contrast < 4.5:1
- ❌ Missing focus states for keyboard navigation
- ❌ Not respecting `prefers-reduced-motion`

---

## Pre-delivery Checklist

- [ ] No emojis as icons (use SVG: Heroicons/Lucide)
- [ ] `cursor-pointer` on all clickable elements
- [ ] Hover states with smooth transitions (150-300ms)
- [ ] Light mode: text contrast 4.5:1 minimum
- [ ] Focus states visible for keyboard nav
- [ ] `prefers-reduced-motion` respected
- [ ] Responsive: 375px, 768px, 1024px, 1440px
